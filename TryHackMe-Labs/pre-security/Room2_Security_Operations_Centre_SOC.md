 Security Operations Centre (SOC)

## What is a SOC?
- Central hub for monitoring and defending an organisation.
- Operates 24/7 with security professionals.

## SOC Tasks
- Monitor security alerts.
- Investigate anomalies.
- Respond to incidents.
- Improve overall security defenses.

## SIEM (Security Information and Event Management)
- Centralized system collecting logs from endpoints, servers, firewalls.
- Correlates events for easier threat detection.
- Provides insight into organisational activity for SOC analysts.
