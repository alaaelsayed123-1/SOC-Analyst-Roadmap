Room 5: Ping and ICMP

## What is Ping?
- Tool to test connectivity and reliability between devices.
- Uses **ICMP (Internet Control Message Protocol)**.

## How Ping Works
1. ICMP Echo Request sent to target.
2. ICMP Echo Reply received.
3. Measures round-trip time (latency) and packet loss.

## Practical Task
- Ping 8.8.8.8 (Google DNS) on the lab.
- Returns ICMP replies and a **flag**.
