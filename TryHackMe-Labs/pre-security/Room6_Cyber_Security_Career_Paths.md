Room 6: Cyber Security Career Paths

## Security Analyst (Blue Team)
- Monitor systems and networks.
- Investigate alerts.
- Respond to incidents.
- Entry-level and high demand role.

## Security Engineer
- Design and maintain security systems.
- Manage IDS, firewalls, and controls.
- Assess risk and vulnerabilities.
- Architect role in defensive security.

## Penetration Tester (Red Team)
- Test systems by safely attempting to break in.
- Identify vulnerabilities.
- Report findings and help prevent real attacks.
- Pathway to offensive security and red teaming.

## Summary
- Rooms provide: SOC experience, networking knowledge, defensive security fundamentals, and career insights.
- Prepares for entry-level cyber security roles or internships.
