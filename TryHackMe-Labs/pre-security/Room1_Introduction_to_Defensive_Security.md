 Introduction to Defensive Security

## What is Defensive Security?
- Focuses on protecting systems, networks, and data from cyber attacks.
- Detect, prevent, and respond to malicious activity.
- Also called the **Blue Team**.

## Defence in Depth
- Multiple layers of protection to reduce risk.
- Examples: Employee Training, Firewalls, IDS, Security Policies.

## Employee Training
- Helps staff recognize phishing and malicious activity.
- Reduces human error in security breaches.

## Intrusion Detection Systems (IDS)
- Acts like security cameras for the network.
- Monitors traffic and alerts suspicious behavior.

## Firewalls
- Acts as a security guard at the network boundary.
- Controls allowed and blocked traffic.

## Security Policies
- Rules for secure system usage.
- Includes password policies, blocked websites, restricted access.
